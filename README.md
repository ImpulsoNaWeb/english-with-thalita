# English with Thalita

Painel Administrativo para gerenciamento do site English with Thalita.

## Tecnologias
- Laravel 11
- Filament v3
- Livewire 3
- SQLite
- Tailwind CSS

## Configuração de Produção (VPS)
O projeto está configurado para utilizar **Trusted Proxies** para detecção automática de HTTPS quando atrás de um proxy reverso.
